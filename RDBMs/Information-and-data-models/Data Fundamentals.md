# Types Of Data

## **Structured**

- Rigid format (schema)
- Rows and Columns

## **Semi-Structured**

- Mix of Structured and Un-structured data
- Hierarchy using tags and meta-data

## Un-structured

- Cannot be organized into tabular format for storage in RDBMS
